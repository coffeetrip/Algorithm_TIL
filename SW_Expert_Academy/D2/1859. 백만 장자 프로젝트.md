```
# 주어진 가격의 범위
0 ~n-1(n일)

# 사는 것을 고려하는 범위(마지막날 제외)
i: 0~n-2
	# 파는 것을 고려한느 범위
	j: i+1 ~n-1
		# i날에 사서 j날에 팔 때 최대 이익을 찾음.
		# 이익이 없으면 사지 않음
```

```
          2 5 : 3의 오른쪽 숫자들. 가장 큰 수 5
```

```python
    money = 0
    while True:
        if num:
            max_num = max(num)
            i = num.index(max_num)
        else:
            break
        for c in num[:i]:
            money += max_num-c
        num = num[i+1:]
    print(f"#{tc+1} {money}")
```
