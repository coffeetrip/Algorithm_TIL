### 풀이

```
수평에서 수직이동
인덱스가 증가하는 방향 +1
길이감소

수직에서 수평이동
인덱스가 감소하는 방향 -1
길이 일정
```

```
NxN 달팽이 배열

초기값 : 이동거리K = N, 이동방향 dir = 1
NxN모든 칸이 채워질 때까지
(1) 수평 이동 (+1).
(1)-(2) 이동 거리 1 감소 N = N-1
k==0이 되면 중단
(2) 수직 이동 (+1) 후 이동 방향 반전. dir *= -1
(1) 반복
```

```python
T = int(input())
for tc in range(T):
    c = 1  # 배열 안 숫자
    N = int(input())
    k = N
    dir = 1
    arr = [[0]*N for _ in range(N)]

    i = 0  # 시작 칸의 인덱스
    j = -1  # 현재 위치로 부터 w번 이동
    while 1:
        for h in range(k):  # 수평이동
            j += dir
            arr[i][j] = c    
            c += 1
        k -= 1     # 이동거리 감소
        if k == 0:   # 이동 거리가 0이면 중단
            break
        for v in range(k):   # 수직이동
            i += dir
            arr[i][j] = c
            c += 1
        dir *= -1    # 수직-> 수평으로 바뀔 때 인덱스 증감 바꾸기

    print(f'#{tc+1}')
    for r in range(N):
        for c in range(N):
            print(arr[r][c], end=" ")
        print()
```

### 탐색

```
# 탐색을 이용한 달팽이
1) 오른쪽부터 시계방향으로 0,1,2,3방향이라 정함
2) 0번 방향으로 설정, 맨 왼쪽 윗칸을 탐색 위치로 정함
3) 설정 방향으로 탐색 시작
4) 현재 방향의 마지막 칸이거나, 다음 칸에 값이 있으면 다음 방향으로 변경
5) NxN칸 이내면 3)반복
6) 더 이상 남은 칸이 없으면 종료
```

```python
di = [0, 1, 0, -1]
dj = [1, 0, -1, 0]

T = int(input())
for tc in range(1, T+1):
    N = int(input())
    arr = [[0]*N for _ in range(N)]

    dir = 0  # di,dj를 이동
    i = 0  # 현재칸 좌표
    j = 0
    k = 1  # 칸에 기록할 값
    while k <= N**2:  # 아직 NXN 칸 이내면
        arr[i][j] = k  # 현재칸에 값을 쓰고
        k += 1  # 다음 칸에 쓸 값 결정
        # 다음 칸을 결정, 배열을 벗어나지 않고 비어있어야 함
        # 현재방향으로 다음 칸을 계산할 지, 다음 방향으로 계산할 지 결정
        ni = i + di[dir]
        nj = j + dj[dir]
        if ni>=0 and ni<N and  nj>=0 and nj<N and arr[ni][nj] == 0:  # 벗어나지 않고 값이 없으면
            i, j = ni, nj
        else:
            dir = (dir + 1) % 4
            i += di[dir]
            j += dj[dir]
    print(arr)
```

```python
for tc in range(int(input())):
    n = int(input())
    lists = [[0]*n for _ in range(n)]
    x,y= 0,0
    a = n
    num = 1
    for i in range(n//2):
        if n == 1:
            break
        for j in range(a-1):
            lists[x][y+j] = str(num)
            num += 1
        y = y+j+1
        for r in range(a-1):
            lists[x+r][y] = str(num)
            num+=1
        x = x+r+1
        for f in range(a-1):
            lists[x][y-f] = str(num)
            num +=1
        y = y-f-1
        for d in range(a-1):
            lists[x-d][y] = str(num)
            num +=1
        a = a-2
        x,y = x-d, y+1
    if n%2 == 1:
        lists[n//2][n//2] = str(n*n)
    print(f'#{tc+1}')
    for fin in lists:
        print(' '.join(fin))
```
