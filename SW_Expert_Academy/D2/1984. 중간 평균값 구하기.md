# 1984. 중간 평균값 구하기

```python
t = int(input())
for tc in range(t):
    num = list(map(int, input().split()))

    for i in range(len(num)-1, -1, -1):
        for j in range(0, i):
            if num[j] > num[j+1]:
                num[j], num[j+1] = num[j+1], num[j]
    new_num = []
    for k in num[1:len(num)-1]:
        new_num.append(k)

    print(f'#{tc+1} {round(sum(new_num)/len(new_num))}')
```
