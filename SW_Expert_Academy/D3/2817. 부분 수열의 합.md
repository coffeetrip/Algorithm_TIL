### 2817. 부분 수열의 합

```python
T = int(input())
for tc in range(1, T+1):
    N, K = map(int, input().split())  # 개수, 합
    A = list(map(int, input().split()))

    cnt = 0
    for i in range(1, 1<<N):
        temp = []
        for j in range(N):
            if i & (1<<j):
                temp.append(A[j])
        if sum(temp) == K:
            cnt += 1

    print('#{} {}' .format(tc, cnt))
```
