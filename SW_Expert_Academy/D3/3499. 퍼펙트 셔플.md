### 3499. 퍼펙트 셔플

```python
T = int(input())
for tc in range(1, T+1):
    N = int(input())
    card = list(input().split())
    if N % 2:
        left = card[:N//2+1]
        right = card[N//2+1:]
    else:
        left = card[:N//2]
        right = card[N//2:]

    shuffle = []
    while len(shuffle) != len(card):
        shuffle.append(left.pop(0))
        if len(right) != 0:
            shuffle.append(right.pop(0))
        else:
            pass
    print('#{}' .format(tc), end=" ")
    print(*shuffle)
```
