### 1486. 장훈이의 높은 선반

```python
T = int(input())
for tc in range(1, T+1):
    N, B = map(int, input().split())
    num = list(map(int, input().split()))  # 키

    all = []
    for i in range(1, 1<<N):
        temp = 0
        for j in range(N):
            if i&(1<<j):
                temp += num[j]
        if temp >= B:
            all.append(temp)

    minV = 10000000000000
    for i in range(len(all)):
        if minV > all[i]:
            minV = all[i]

    print('#{} {}' .format(tc, minV-B))
```
